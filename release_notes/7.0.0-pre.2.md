# Jasmine 7.0.0-pre.2 Release Notes

This release updates the jasmine-core dependency to 7.0.0-pre.2. See the
[jasmine-core release notes](https://github.com/jasmine/jasmine/blob/main/release_notes/7.0.0-pre.2.md)
for more information.

## Breaking changes

* Redundant configuration properties that were deprecated in 6.3.0 have been 
  removed. Use the corresponding env config properties instead, e.g.
  `env.random` instead of `random`.
* The jsLoader config setting has been removed. The default loader now covers
  most use cases for it. If you find that the default loader still doesn't meet
  your needs, use a custom loader.
* `Jasmine#loadConfig` and `ParallelRunner#loadConfig` are now async.

## New features

* [Custom loaders](https://jasmine.github.io/api/npm/7.0.0-pre.2/Configuration.html#loader)
  allow replacing the built-in module loading logic with your own.

## Supported environments

This version has been tested on Node 20, 22, and 24.
